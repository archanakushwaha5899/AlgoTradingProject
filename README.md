# Algo-Trading Prototype with Logistic Regression

## Objective
Build a Python-based algorithmic trading system combining RSI & SMA crossover logic with a Logistic Regression ML model for predicting next-day stock movements.

## Steps Implemented
1. Data ingestion from Yahoo Finance for NIFTY50 stocks.
2. RSI, SMA, and MACD indicator computation.
3. Rule-based buy/sell signal generation.
4. ML model training and evaluation.
5. Automated comparison and Google Sheets logging.

## Tools Used
Python, Pandas, yfinance, scikit-learn, Matplotlib, gspread

## Results
- RELIANCE achieved highest accuracy (0.625)
- Final summary and plots available in `Comparison_Results`
